VSCode中格式化JSON文件

使用右键菜单

  1. VSCode 窗口右下角的语言模式显示为 JSON
  2. 在编辑器中右键单击，选择 "格式化文档"（Format Document）

注意事项

  • 确保文件的语言模式正确识别为 JSON
  • 如果 JSON 格式有误，格式化可能会失败
  • 可以在设置中配置格式化选项
