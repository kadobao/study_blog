前置工作

检测是否安装了 typescript

tsc --version

安装 typescript

npm list -g typescript

检测是否安装了 nodejs

node -v

检测是否安装了 npm

npm -v

如果没有去 [Node.js 的官方网站地址](Node.js 的官方网站地址) 安装 nodejs。
