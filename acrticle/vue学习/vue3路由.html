XJ2024/10/14еӨ§зәҰ 3 еҲҶй’ҹvueеӯҰд№ vueеӯҰд№ 

дёҖдёӘжҳҜи·Ҝз”ұи§„еҲҷ(index.ts)еңЁpathе“ӘйҮҢе®ҡд№үдәҶеҸӮж•°еә”иҜҘеңЁurlзҡ„йӮЈдёӘдҪҚзҪ®пјҢдёҖдёӘжҳҜи·іиҪ¬й“ҫжҺҘ(дёҖиҲ¬жҳҜеҪ“еүҚз»„д»¶)иҝҷйҮҢдј йҖ’еҸӮж•°пјҢдёҖдёӘжҳҜзӣ®ж Үз»„д»¶пјҢеҶіе®ҡдј йҖ’иҝҮжқҘзҡ„еҸӮж•°еҰӮдҪ•дҪҝз”ЁгҖӮ

и·Ҝз”ұжЁЎејҸпјҡVue Router жҸҗдҫӣдёӨз§Қдё»иҰҒзҡ„и·Ҝз”ұжЁЎејҸпјҡ

  • Hash жЁЎејҸпјҡдҪҝз”Ё URL зҡ„е“ҲеёҢеҖјпјҲ#пјүиҝӣиЎҢи·Ҝз”ұпјҢең°еқҖж ҸзңӢиө·жқҘеғҸ http://example.com/#/aboutгҖӮ
  • History жЁЎејҸпјҡеҲ©з”Ё HTML5 зҡ„ History APIпјҢURL зңӢиө·жқҘжӣҙвҖңе№ІеҮҖвҖқпјҢеҰӮ http://example.com/aboutпјҢжІЎжңү # з¬ҰеҸ·гҖӮ

еҠЁжҖҒз»‘е®ҡдёҺи·Ҝз”ұдј еҸӮ

еҜ№иұЎеҶҷжі•йңҖиҰҒеҠЁжҖҒз»‘е®ҡ JavaScript иЎЁиҫҫејҸпјҢеӣ жӯӨеҝ…йЎ»дҪҝз”Ё v-bindпјҢд№ҹе°ұжҳҜз®ҖеҶҷеҪўејҸ :toгҖӮ
:to иҜӯжі•з”ЁдәҺе°ҶеұһжҖ§еҠЁжҖҒз»‘е®ҡеҲ°иЎЁиҫҫејҸпјҢиҖҢдёҚжҳҜз®ҖеҚ•зҡ„еӯ—з¬ҰдёІпјҢйҖӮз”ЁдәҺдј йҖ’еҠЁжҖҒеҸӮж•°пјҢеҰӮдёҚеҗҢзҡ„ nameгҖӮ

{
  name: 'zhuye',  // и·Ҝз”ұеҗҚеӯ—

}

е®ҡд№үеҠЁжҖҒи·Ҝеҫ„еҸӮж•°

дҪҝз”Ё : е®ҡд№үи·Ҝеҫ„дёӯзҡ„еҠЁжҖҒеҸӮж•°гҖӮдҫӢеҰӮпјҢ/user/:id дёӯзҡ„ :id иЎЁзӨә id жҳҜдёҖдёӘеҠЁжҖҒеҸӮж•°пјҡ

  • дҫӢеҰӮ /user/123 е’Ң /user/456 йғҪдјҡеҢ№й…ҚиҜҘи·Ҝз”ұпјҢе№¶е°Ҷ 123 жҲ– 456 дҪңдёә id зҡ„еҖјдј йҖ’з»ҷзӣ®ж Үз»„д»¶гҖӮ

еҢәеҲ«пјҡи·Ҝеҫ„еҸӮж•°дёҺжҹҘиҜўеҸӮж•°

  • и·Ҝеҫ„еҸӮж•°пјҲparamsпјүпјҡз”ЁдәҺж ҮиҜҶиө„жәҗпјҢзӣҙжҺҘеҪұе“ҚйЎөйқўжҳҫзӨәгҖӮдҫӢеҰӮ /news/1 иЎЁзӨә ID дёә 1 зҡ„ж–°й—»гҖӮ
  • жҹҘиҜўеҸӮж•°пјҲqueryпјүпјҡз”ЁдәҺзӯӣйҖүжҲ–жҺ’еәҸпјҢеҶіе®ҡйЎөйқўзҡ„йҖ»иҫ‘ж“ҚдҪңгҖӮдҫӢеҰӮ /user?id=1&filter=activeгҖӮ

и·Ҝз”ұдј еҸӮжң¬иә«дёҚдјҡиҮӘеҠЁеј•иө·йЎөйқўеҶ…е®№еҸҳеҢ–пјҢз»„д»¶еҰӮдҪ•еӨ„зҗҶиҝҷдәӣеҸӮж•°еҶіе®ҡйЎөйқўеҸҳеҢ–гҖӮдҫӢеҰӮпјҢи·Ҝеҫ„еҸӮж•°еёёз”ЁдәҺж ҮиҜҶе”ҜдёҖиө„жәҗпјҢжҹҘиҜўеҸӮж•°еҲҷз”ЁдәҺеӨҚжқӮйҖ»иҫ‘еӨ„зҗҶгҖӮ

и·Ҝеҫ„е’ҢжҹҘиҜўеҸӮж•°жҳҜжІЎжңүеҢәеҲ«зҡ„пјҢеҸӘжҳҜи·Ҝеҫ„еҸӮж•°еҸӘжҳҜз®ҖеҚ•зҡ„йҖ»иҫ‘(дёҖиҲ¬е°ұжҳҜзӣҙжҺҘдҪҝз”ЁдҪ дј иҝҮжқҘзҡ„)пјҢжҹҘиҜўеҸӮж•°дёҖиҲ¬жҜ”и·Ҝеҫ„еҸӮж•°зҡ„йҖ»иҫ‘еӨҚжқӮгҖӮ

и·Ҝеҫ„еҸӮж•°жҳҜжҳҫзӨәйЎөйқўпјҢжҹҘиҜўеҸӮж•°жҳҜжңүйҖ»иҫ‘пјҢзӣ®ж Үз»„д»¶еҶҷзҡ„йҖ»иҫ‘д»Јз ҒеҶіе®ҡеҰӮдҪ•еӨ„зҗҶжҹҘиҜўеҸӮж•°пјҢеҰӮдҪ•еј•иө·йЎөйқўеҸҳеҢ–гҖӮ

дҪҝз”Ё router.push дј йҖ’и·Ҝеҫ„еҸӮж•°е№¶жҺҘж”¶

еңЁ Vue 3 дёӯпјҢи·Ҝеҫ„еҸӮж•°йҖҡиҝҮ router.push дј йҖ’пјҢеңЁзӣ®ж Үз»„д»¶дёӯдҪҝз”Ё useRoute й’©еӯҗжҺҘж”¶гҖӮ

1. й…ҚзҪ® Vue Router

import { createRouter, createWebHistory } from 'vue-router';
import Home from '../views/Home.vue';
import User from '../views/User.vue';

const routes = [
  { path: '/', name: 'Home', component: Home },
  { path: '/user/:id', name: 'User', component: User }
];

const router = createRouter({
  history: createWebHistory(),
  routes
});

export default router;
