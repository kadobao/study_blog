C# 中常见整数类型与 Windows API/传统数据类型（如 WORD、DWORD）的对照表

Windows / 传统类型C# 类型.NET 类型名有符号位数范围（近似十进制）说明
WORDushortSystem.UInt16否160 到 65,535无符号 16 位整数
DWORDuintSystem.UInt32否320 到 4,294,967,295无符号 32 位整数
shortshortSystem.Int16是16-32,768 到 32,767有符号 16 位整数
ushortushortSystem.UInt16否160 到 65,535同 WORD
intintSystem.Int32是32-2,147,483,648 到 2,147,483,647有符号 32 位整数
uintuintSystem.UInt32否320 到 4,294,967,295同 DWORD
longlongSystem.Int64是64±9.22×10¹⁸有符号 64 位整数
ulongulongSystem.UInt64否640 到 1.84×10¹⁹无符号 64 位整数
float / SinglefloatSystem.Single是32±1.5×10⁻⁴⁵ 到 ±3.4×10³⁸（约 7 位有效数字）IEEE 754 单精度浮点
double / DoubledoubleSystem.Double是64±5.0×10⁻³²⁴ 到 ±1.7×10³⁰⁸（约 15–17 位有效数字）IEEE 754 双精度浮点
boolboolSystem.Boolean—通常为 8（1 字节）*true 或 false布尔值
