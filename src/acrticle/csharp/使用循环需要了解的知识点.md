## 循环需要了解的关键字

| 关键字 | 作用 |
|--------|------|
| `continue` | 跳过当前循环，进入下一次循环 |
| `break` | 结束循环（是结束循环，而不只是跳过当前循环） |
| `return` | 结束当前方法，返回调用者 |

## 解决循环里面网络请求断连的异常处理

### 问题描述
`while` 循环的每一次迭代都有可能会发生连接断开的情况，所以我们需要在循环里面进行异常处理。

### 解决方案
`while` 循环的某一次连接断开时，进行重连。

**实现步骤：**

1. 先在 `while` 循环外设置一个 `bool` 变量用于标志是否进行连接
2. 使用 `try-catch` 捕捉到异常，将连接变量设置为 `false`
3. 下一次循环就会进行重连

### 示例代码

```csharp
bool isConnected = false;

while (true)
{
    try
    {
        if (!isConnected)
        {
            // 进行重连操作
            Connect();
            isConnected = true;
        }
        
        // 业务逻辑
        DoSomething();
    }
    catch (Exception ex)
    {
        // 连接断开，设置标志为 false，下次循环进行重连
        isConnected = false;
        Console.WriteLine($"连接断开: {ex.Message}");
    }
}
```
