### 前置工作

检测是否安装了 typescript

```powershell
tsc --version
```

安装 typescript

```powershell
npm list -g typescript
```

检测是否安装了 nodejs
```powershell
node -v
```

检测是否安装了 npm

```powershell
npm -v
```

如果没有去 [Node.js 的官方网站地址](Node.js 的官方网站地址) 安装 nodejs。
