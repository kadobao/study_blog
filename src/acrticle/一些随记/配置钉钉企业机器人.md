## 前置准备

先去钉钉开发者后台：https://open-dev.dingtalk.com/?spm=ding_open_doc.document.0.0.55467681Wlktsg

## 创建应用

1. 单击**应用开发** > **企业内部应用** > **钉钉应用** > **创建应用**

   ![创建钉钉企业机器人_1](/assets/images/创建钉钉企业机器人_1.png)

2. 填写好应用信息，然后点击**保存**

   ![创建钉钉企业机器人_2](/assets/images/创建钉钉企业机器人_2.png)

## 配置机器人

1. 点击刚才新创建的应用，点击进入应用详情页
2. 在左侧侧边栏点击**机器人**
3. 启用机器人配置，配置好机器人信息
4. 将**消息接收模式**选择为 `Stream` 模式

   ![创建钉钉企业机器人_3](/assets/images/创建钉钉企业机器人_3.png)

## 发布应用

1. 点击**查看版本详情**跳转到**版本管理与发布**页面
2. 点击**编辑**，填写好版本信息
3. 点击**发布**按钮

   ![创建钉钉企业机器人_4](/assets/images/创建钉钉企业机器人_4.png)

   ![创建钉钉企业机器人_5](/assets/images/创建钉钉企业机器人_5.png)

## 获取凭证

点击**凭证与基础信息**，复制 `Client ID` 和 `Client Secret`

![创建钉钉企业机器人_6](/assets/images/创建钉钉企业机器人_6.png)

## 使用 Python 接收消息

以下示例展示了如何使用 Python 接收钉钉企业机器人消息和回复消息：

```python
"""
钉钉机器人服务 - 接收并处理消息

本文件实现了一个简单的钉钉聊天机器人服务，主要功能：
1. 通过 Stream 模式连接钉钉开放平台
2. 接收用户发送的文本消息
3. 将消息原样回复给用户

参考文档：https://opensource.dingtalk.com/developerpedia/docs/explore/tutorials/stream/bot/python/build-bot
"""

from dingtalk_stream import AckMessage
import dingtalk_stream

# ============================================
# 👇 直接在这里填入你的应用凭证
# ============================================
# Client ID（App Key）：从钉钉开放平台获取的应用唯一标识
CLIENT_ID = "Your_Client ID"

# Client Secret（App Secret）：从钉钉开放平台获取的应用密钥
CLIENT_SECRET = "Your_Client Secret"
# ============================================


class EchoTextHandler(dingtalk_stream.ChatbotHandler):
    """
    消息处理器类 - 继承自 ChatbotHandler

    该类负责处理接收到的聊天机器人消息，并执行相应的回复操作
    """

    def __init__(self):
        # 调用父类的构造函数，初始化处理器
        super(dingtalk_stream.ChatbotHandler, self).__init__()

    async def process(self, callback: dingtalk_stream.CallbackMessage):
        """
        处理接收到的消息

        Args:
            callback: 钉钉推送的回调消息对象，包含消息的完整数据

        Returns:
            AckMessage.STATUS_OK: 确认消息已成功处理
            'OK': 处理成功的状态描述
        """
        # 从回调数据中解析出聊天机器人消息
        incoming_message = dingtalk_stream.ChatbotMessage.from_dict(callback.data)

        # 获取消息文本内容并去除首尾空白
        text = incoming_message.text.content.strip()

        # 打印接收到的消息到控制台
        print(f"收到消息: {text}")

        # 将原消息回复给用户（echo 回复）
        self.reply_text(text, incoming_message)

        # 返回成功状态，告知钉钉服务器消息已处理
        return AckMessage.STATUS_OK, 'OK'


def main():
    """
    主函数 - 启动机器人服务

    流程：
    1. 使用 Client ID 和 Client Secret 创建凭证对象
    2. 创建 DingTalkStreamClient 客户端
    3. 注册消息回调处理器
    4. 启动服务，开始监听消息
    """
    # 创建钉钉Stream连接凭证，包含应用的身份信息
    credential = dingtalk_stream.Credential(CLIENT_ID, CLIENT_SECRET)

    # 创建 DingTalk Stream 客户端实例
    client = dingtalk_stream.DingTalkStreamClient(credential)

    # 注册机器人消息的回调处理器
    # ChatbotMessage.TOPIC 表示监听聊天机器人类型的消息
    client.register_callback_handler(
        dingtalk_stream.chatbot.ChatbotMessage.TOPIC,
        EchoTextHandler()
    )

    # 输出启动提示信息
    print("机器人服务已启动，等待消息中...")

    # 启动客户端，进入永久循环，持续接收和处理消息
    client.start_forever()


# 程序入口点
if __name__ == '__main__':
    main()
```
