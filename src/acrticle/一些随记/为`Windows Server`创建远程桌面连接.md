# 为 Windows Server 创建远程桌面连接

本教程将指导您如何为 Windows Server 配置远程桌面连接功能。

## 步骤一：开启远程桌面权限

1. 在"此电脑"上右键选择`属性`
2. 点击"远程"部分
3. 选择`允许远程连接此电脑`

![开启远程权限_1](/assets/images/开启远程权限_1.png)

![开启远程权限_2](/assets/images/开启远程权限_2.png)

## 步骤二：创建远程访问用户

1. 右键选择`管理`
2. 进入`本地用户和组\用户`
3. 右键选择`新用户`
4. 依次填写用户信息
5. 选择`永不过期`
6. 右键新创建的用户，重新输入密码

![创建远程访问的用户_1](/assets/images/创建远程访问的用户_1.png)

![创建远程访问的用户_2](/assets/images/创建远程访问的用户_2.png)

![创建远程访问的用户_3](/assets/images/创建远程访问的用户_3.png)

## 步骤三：将用户添加到远程桌面用户组

1. 双击新创建的用户
2. 点击`隶属于`
3. 点击`添加`
4. 在对象名称来源框中输入 `remote desktop users`
5. 点击`检查名称`
6. 点击`应用`、`确定`

![将用户添加到远程桌面用户组_1](/assets/images/将用户隶属于远程桌面用户组.png)

![将用户添加到远程桌面用户组_2](/assets/images/将用户隶属于远程桌面用户组_2.png)

![将用户添加到远程桌面用户组_3](/assets/images/将用户隶属于远程桌面用户组_3.png)

## 完成配置

完成以上步骤后，您就可以使用远程桌面连接到这台 Windows Server 了。

> **注意**：确保防火墙设置允许远程桌面连接。
