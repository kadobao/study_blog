下载sql server安装包

[下载地址](https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads?spm=a2ty_o01.29997173.0.0.77455171Gbj2kX)

下载`SQL Server 2022 Developer`


双击安装包之后选择`自定义安装`，

然后设置好安装路径，点击`安装`按钮，等待安装完成。

安装完成之后，弹出的弹窗点击`安装`，选择`全新SQLServer独立安装或向现有安装添加功能`


指定可用版本选择`Developer`，然后点击`下一步`

然后点击`我接受协议`，然后点击`下一步`
