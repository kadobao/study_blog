C# 中常见整数类型与 Windows API/传统数据类型（如 WORD、DWORD）的对照表

| Windows / 传统类型 | C# 类型        | .NET 类型名       | 有符号 | 位数 | 范围（近似十进制）                                 | 说明 |
|--------------------|----------------|--------------------|--------|------|----------------------------------------------------|------|
| **WORD**           | `ushort`       | `System.UInt16`    | 否     | 16   | 0 到 65,535                                        | 无符号 16 位整数 |
| **DWORD**          | `uint`         | `System.UInt32`    | 否     | 32   | 0 到 4,294,967,295                                 | 无符号 32 位整数 |
| **short**          | `short`        | `System.Int16`     | 是     | 16   | -32,768 到 32,767                                  | 有符号 16 位整数 |
| **ushort**         | `ushort`       | `System.UInt16`    | 否     | 16   | 0 到 65,535                                        | 同 WORD |
| **int**            | `int`          | `System.Int32`     | 是     | 32   | -2,147,483,648 到 2,147,483,647                    | 有符号 32 位整数 |
| **uint**           | `uint`         | `System.UInt32`    | 否     | 32   | 0 到 4,294,967,295                                 | 同 DWORD |
| **long**           | `long`         | `System.Int64`     | 是     | 64   | ±9.22×10¹⁸                                         | 有符号 64 位整数 |
| **ulong**          | `ulong`        | `System.UInt64`    | 否     | 64   | 0 到 1.84×10¹⁹                                     | 无符号 64 位整数 |
| **float** / **Single** | `float`    | `System.Single`    | 是     | 32   | ±1.5×10⁻⁴⁵ 到 ±3.4×10³⁸（约 7 位有效数字）        | IEEE 754 单精度浮点 |
| **double** / **Double** | `double`  | `System.Double`    | 是     | 64   | ±5.0×10⁻³²⁴ 到 ±1.7×10³⁰⁸（约 15–17 位有效数字） | IEEE 754 双精度浮点 |
| **bool**           | `bool`         | `System.Boolean`   | —      | 通常为 8（1 字节）* | `true` 或 `false`                              | 布尔值 |
