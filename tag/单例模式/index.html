单例模式

单例模式是一种常用的软件设计模式，它保证一个类只有一个实例，并提供一个全局访问点。以下是实现线程安全的单例模式的步骤：

步骤 1: 创建类并设置私有构造函数

将类的构造函数设为私有，防止外部直接实例化。

public class Log
{
    // 私有构造函数，防止外部实例化
    private Log()
    {
        Init();
    }
}
